Introduction à Bootstrap 4

Cette page fonctionne avec Bootstrap 4 en CDN.

Qu'est ce que Bootstrap ?

  • Bootstrap est un framework CSS ("cadriciel").C'est un cadre de travail.
  • C'est une librairie côté client (front end).
  • Bootstrap aide à mettre en forme les pages web.
  • C'est une librairie dans laquelle il y a des ressources CSS mais il y a également des composants HTML 5 et JavaScript.
  • Framework bas" sur sass, du CSS avancé et normalize.
  • Il comporte également une grille de mise en page.

Quel est l'intérêt de Bootstrap ?

  • Bootstrap va permettre d'uniformiser pour les navigateurs.
  • Il va permettre de gagner du temps car nous n'avons pas tout à concevoir.
  • Des styles identiques
  • Une grille responsive
  • Et plein d'autres éléments
  • C'est une ressource gratuite
  • Documentation très précise pour apprendre à utiliser !
  • Très bonne gestion des media queries

Des inconvénients ?

Bootstrap suppose un temps d'apprentissage , c'est complexe.
